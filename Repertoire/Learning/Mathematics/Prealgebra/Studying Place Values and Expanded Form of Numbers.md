## Place Value
**Place-Value:** Position of each digit in a number
1 784 653 201 459

Consider the number 1 784 653 201 459.

| **Place-Value**   | **Number** |
| ----------------- | ---------- |
| Trillions         | 1          |
| Hundred-Billions  | 7          |
| Ten-Billions      | 8          |
| Billions          | 4          |
| Hundred-Millions  | 6          |
| Ten-Millions      | 5          |
| Millions          | 3          |
| Hundred-Thousands | 2          |
| Ten-Thousands     | 0          |
| Thousands         | 1          |
| Hundreds          | 4          |
| Tens              | 5          |
| Ones              | 9           |
